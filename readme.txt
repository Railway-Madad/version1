🚆 Railway Madad App

Railway Madad is a complaint management system for railway passengers. 
It allows users to register complaints, track status, and enables admins and staff to manage and resolve complaints efficiently.

📌 Features
👥 User Features
✅ User registration & login
✅ Enter PNR number for verification
✅ File a complaint with details
✅ View complaint status & history

🛠️ Admin Features
✅ View all complaints
✅ Assign complaints to staff
✅ Update complaint status (Pending, In Progress, Resolved)
✅ Generate reports for analysis

👷 Staff Features
✅ View assigned complaints
✅ Update progress on complaints
✅ Mark complaints as resolved

🏗️ Tech Stack
🔹 Frontend: EJS, CSS, JavaScript
🔹 Backend: Node.js, Express
🔹 Database: In-memory JSON, SQLite
🔹 Authentication: Normal authentication
🔹 Hosting: Localhost
